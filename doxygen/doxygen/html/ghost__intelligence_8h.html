Assign a quality value to all the nodes in the openNodes map.

Parameters
nodesa vector of all the nodes in the map
openNodesa map that contains all the nodes as keys and their quality as value
pacmanPosthe position of the pacman This function assigns a quality value to all the nodes in the openNodes map based on the distance between the node and the pacman. The quality value is the sum of the absolute differences between the x and y coordinates of the node and the pacman. The higher the quality, the farther the node is from the pacman.
